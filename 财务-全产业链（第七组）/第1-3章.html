第一章

管理会计理论基础

系统学习管理会计的核心方法，为成本管控提供理论武器

1.1 目标成本法（Target Costing）

核心概念
目标成本法是一种以市场为导向的成本管理方法，其基本逻辑是通过市场价格倒推企业必须达到的成本水平。这种方法将传统的"成本加成定价"思维转变为"市场倒逼成本"思维，特别适用于竞争激烈、价格透明的轴承行业。
目标成本 = 市场竞争价格 - 目标利润
❌ 传统成本加成法
价格 = 成本 + 利润

特点：企业内部驱动，先算成本再定价格

弊端：可能脱离市场，价格缺乏竞争力

风险：在买方市场中难以销售

✓ 目标成本法
成本 = 价格 - 利润

特点：市场倒逼驱动，根据市场价格确定成本目标

优势：确保产品具有市场竞争力

核心：成本必须服从市场

实施五步骤
步骤1：市场价格调研
① 收集主要竞争对手报价信息
② 了解客户期望价格和接受范围
③ 分析市场价格趋势和波动规律
④ 确定具有竞争力的市场价格定位
↓
步骤2：确定目标利润
① 根据企业战略规划设定利润目标
② 考虑股东回报要求和行业平均水平
③ 综合评估风险和投资回报期
④ 确定合理的目标利润率（如15%）
↓
步骤3：计算目标成本
目标成本 = 市场价格 × (1 - 目标利润率)
确保目标成本具有可达性和挑战性
↓
步骤4：成本差异分析
成本差距 = 实际成本 - 目标成本
差异率 = 成本差距 / 目标成本 × 100%
识别差距较大的成本项目
↓
步骤5：制定降本措施
工艺优化、采购降本、材料替代、效率提升等
💡 案例：某轴承产品目标成本计算

基础数据：

  • 市场竞争价格：100元/件
  • 目标利润率：15%
  • 当前实际成本：92元/件
  • 年销量：10万件
目标成本 = 100 × (1-15%) = 85元/件 成本差距 = 92 - 85 = 7元/件 需降本率 = 7/92 × 100% = 7.61% 效益分析： 当前利润：(100-92) × 10万 = 80万元 目标利润：(100-85) × 10万 = 150万元 利润提升：70万元（+87.5%）

降本措施（7元分解）：

  • 材料成本：采购降价+国产替代+提高利用率 → 节约5元
  • 人工成本：工艺优化+效率提升 → 节约1元
  • 制造费用：OEE提升+能源管理 → 节约1元
⚠️ 应用要点：
  • 适用于竞争激烈、价格透明的市场（轴承行业特别适合）
  • 需要跨部门协作，建立成本管控联动机制
  • 强调事前控制，在设计阶段就考虑成本目标
  • 降本不能以牺牲质量为代价

1.2 作业成本法（ABC）

核心理念
产品消耗作业，作业消耗资源

作业成本法通过识别作业链，将成本更准确地分配到产品上，避免传统方法的"成本扭曲"问题。

传统成本法 vs ABC对比
对比维度 传统成本法 作业成本法
分配基础 单一（如工时） 多元（基于作业动因）
分配逻辑 资源 → 产品 资源 → 作业 → 产品
准确性 低（易产生成本扭曲） 高（真实反映资源消耗）
适用场景 间接费用<20%，产品单一 间接费用>30%，产品多样化
信息价值 财务报告 支持决策和成本改进
ABC实施五步法
1. 识别作业

分析流程，识别关键作业活动

轴承企业典型作业：

  • 采购订单处理
  • 机器加工
  • 换模换线
  • 质量检验
2. 归集成本

将资源成本归集到作业成本池

示例：

  • 采购部工资 → 采购作业
  • 设备折旧 → 加工作业
3. 选择动因

确定驱动成本的因素

示例：

  • 采购 → 订单次数
  • 加工 → 机器工时
  • 检验 → 检验批次
4. 计算成本率
成本率 = 成本池 / 动因总量

示例：

200元/次订单

5. 分配到产品
成本 = Σ(成本率 × 动因量)

示例：

200元 × 50次 = 10,000元

💡 完整案例：ABC法揭示成本真相

背景：某轴承企业生产A、B两种产品，制造费用240万元

项目 产品A（大批量） 产品B（小批量）
年产量 10,000件 2,000件
直接工时 20,000小时 10,000小时

传统方法（按工时分配）：

分配率 = 240万 / 30000小时 = 80元/小时 产品A：80×20000 = 160万 → 单位成本760元 产品B：80×10000 = 80万 → 单位成本1,350元

ABC方法（按作业分配）：

作业 成本池 动因 产品A 产品B
机器加工 120万 机时 20000h 10000h
生产准备 60万 批次 40批 80批↑
质量检验 40万 检验次数 100次 100次
物料搬运 20万 搬运次数 200次 200次

ABC计算结果：

产品A制造费用：130万 → 单位成本730元（比传统低30元） 产品B制造费用：110万 → 单位成本1,500元（比传统高150元！） 核心发现： 产品B虽然工时少，但频繁换模、检验多， 消耗大量间接资源，传统方法严重低估了成本！

管理启示：

  • 产品A：成本被高估，可适当降价提升竞争力
  • 产品B：真实成本高，需提价或优化生产方式（增加批量、减少换模）
  • 改进方向：针对产品B，重点优化生产准备作业
✓ ABC法应用价值：
  • 精准成本核算，消除成本扭曲
  • 识别高成本作业，明确改进重点
  • 支持定价决策和产品组合优化
  • 为工艺优化提供数据支持

1.3 价值工程（VE）

核心公式
价值(V) = 功能(F) / 成本(C)
价值工程的目标是以最低的生命周期成本，可靠地实现产品的必要功能，从而提高产品价值。
提升价值的四种途径
途径一（最常用）

功能不变，降低成本

F → / C ↓ = V ↑

通过工艺优化、材料替代等降低成本

途径二

成本略增，功能大幅提升

F ↑↑ / C ↑ = V ↑

投入合理成本，显著改进性能

途径三（理想）

功能提升，成本降低

F ↑ / C ↓ = V ↑↑

通过技术创新实现双重优化

途径四（最佳）

功能大幅提升，成本略降

F ↑↑ / C ↓ = V ↑↑↑

需要重大创新突破

VE实施五步工作计划
第一阶段：信息收集
• 收集产品图纸、工艺文件、成本数据
• 了解客户需求和使用环境
• 收集竞争对手产品信息
↓
第二阶段：功能分析
• 功能定义：用"动词+名词"描述（如"传递扭矩"）
• 功能分类：基本功能vs辅助功能
• 功能评价：识别成本过高的功能
↓
第三阶段：创新构思
• 头脑风暴：提出多种改进方案
• 方案分类：工序简化、材料替代、结构优化
↓
第四阶段：方案评价
• 技术可行性、经济性、风险评估
• 用评分矩阵选出最优方案
↓
第五阶段：实施验证
• 制定实施计划、试验验证、推广应用
轴承降本中的VE应用
优化方向 VE方法 具体措施 预期效果
材料优化 材料替代 进口钢材→国产优质钢
特殊合金→普通合金		 成本降低15-20%

多次热处理→一次完成		 工时减少20-30%

1.4 标准成本法

核心概念
标准成本是在正常生产条件下，通过科学测定制定的单位产品成本标准。标准成本法的核心是差异分析，通过对比实际成本与标准成本，找出差异并分析原因。
标准成本 = 直接材料标准成本 + 直接人工标准成本 + 制造费用标准成本
三大差异分析（重点）
一、材料成本差异分析
材料成本差异 = 实际成本 - 标准成本
= (实际用量×实际价格) - (标准用量×标准价格)

分解为：

价格差异 = 实际用量×(实际价-标准价)

责任：采购部

用量差异 = 标准价×(实际量-标准量)

责任：生产部、工艺部

二、人工成本差异分析
人工差异 = (实际工时×实际工资率) - (标准工时×标准工资率)

分解为：

工资率差异 = 实际工时×(实际率-标准率)

责任：人力资源部

效率差异 = 标准率×(实际时-标准时)

责任：生产部

三、制造费用差异分析（三因素法）
1. 耗费差异 = 实际制造费用 - 预算制造费用 → 反映费用控制水平 2. 效率差异 = 预算制造费用 - (标准分配率×实际工时) → 反映工时利用效率 3. 能量差异 = (标准分配率×实际工时) - 标准制造费用 → 反映产能利用程度
💡 差异分析实战案例

标准成本：材料10kg×50元=500元

实际成本：材料10.5kg×52元=546元（100件）

总差异 = 546×100 - 500×100 = 4,600元（不利） 价格差异 = 1050×(52-50) = 2,100元（不利） → 采购价格上涨，采购部需说明原因 用量差异 = 50×(1050-1000) = 2,500元（不利） → 材料超用5%，生产部需查找原因 （可能：工艺问题、操作不当、质量问题）

1.5 本量利分析（CVP）

核心公式
利润 = (单价 - 单位变动成本) × 销量 - 固定成本
= 单位贡献毛益 × 销量 - 固定成本
关键指标
贡献毛益
= 收入 - 变动成本

用于补偿固定成本和形成利润

贡献毛益率
= 贡献毛益 / 收入

每1元收入的贡献能力

盈亏平衡点
= 固定成本 / 单位贡献

不亏不盈的最低销量

安全边际率
= 安全边际 / 实际销量

反映经营安全程度

💡 降本3%的利润放大效应

基础数据：售价100元，变动成本60元，固定成本200万，销量10万件

当前情况： 贡献毛益 = 100-60 = 40元 利润 = 40×10万 - 200万 = 200万元 降本3%后： 新变动成本 = 60×(1-3%) = 58.2元 新贡献毛益 = 100-58.2 = 41.8元 新利润 = 41.8×10万 - 200万 = 218万元 效果： 成本降低3%，利润增加9%（放大3倍！） 盈亏平衡点从5万降至4.78万件

2.1 成本核算方法

成本核算的基本要求
四项费用界限划分：
  1. 收益性支出 vs 资本性支出
    受益期<1年→费用化；>1年→资本化
  2. 产品成本 vs 期间费用
    生产相关→产品成本；非生产→期间费用
  3. 本期成本 vs 下期成本
    按权责发生制原则划分
  4. 不同产品之间的成本
    准确归集和合理分配
三种成本计算方法对比
方法 适用生产类型 成本对象 计算期 轴承行业适用性
品种法 大量大批单步骤
（发电、采掘）		 产品品种 定期（月） ❌ 不适用
（多步骤）
分批法 单件小批
（重型机械、船舶）		 产品批别 不定期（完工时） △ 部分适用
（定制产品）
分步法 大量大批多步骤
（纺织、冶金、轴承）		 产品及各步骤 定期（月） ✓ 最适合
（多步骤生产）
分步法详解（轴承行业推荐）
逐步结转分步法

特点：半成品成本随实物转移而结转

优点：能提供各步骤半成品成本

缺点：成本结转工作量大

适用：半成品有对外销售需求

平行结转分步法

特点：各步骤只计本步骤成本，最后汇总

优点：简化结转，避免重复

缺点：不提供半成品成本

适用：不需要半成品成本数据

2.2 料工费分析

料工费三要素
产品成本 = 直接材料（料）+ 直接人工（工）+ 制造费用（费）
直接材料（料）
  • 主要材料：钢材、合金
  • 辅助材料：滚珠、保持架
  • 包装材料：纸箱、防锈油
材料费率 = 直接材料 / 总成本 × 100%
直接人工（工）
  • 生产工人基本工资
  • 奖金、津贴
  • 社保公积金
人工费率 = 直接人工 / 总成本 × 100%
制造费用（费）
  • 间接人工、设备折旧
  • 水电费、维修费
  • 低值易耗品、办公费
制造费用率 = 制造费用 / 总成本 × 100%

成本管理理论

全面成本管理体系与高级成本管理方法

3.1 全面成本管理

全面成本管理的"三全"
全员参与

从高层到基层，从采购到销售，人人都是成本控制主体

关键：建立全员成本意识和责任体系

全过程管理

从产品研发到售后服务，全生命周期成本管理

关键：事前、事中、事后控制相结合

全方位控制

涵盖采购、生产、质量、物流、销售等各环节

关键：跨部门协同，系统优化

成本管理的三个层次
战略层：成本战略定位
确定企业成本竞争战略（成本领先 vs 差异化）
瓦轴选择：成本优势+质量保证（"质量上上、价格中上"）
↓ 指导
管理层：成本管理职能
成本预测、决策、计划、控制、分析、考核
建立目标成本体系，实施全链路成本联动模型
↓ 支撑
执行层：日常成本核算
成本数据采集、成本计算、费用审批、日常控制
标准成本管理，差异分析与纠偏
全生命周期成本管理
产品全生命周期成本 = 前期成本 + 生产成本 + 后期成本

3.2 约束理论（TOC）

核心思想
"任何系统至少存在一个约束，否则它就能产生无限的产出。"
"系统的产出由其最薄弱的环节决定。"（木桶原理）

以色列物理学家艾利·高德拉特（Eliyahu M. Goldratt）在其著作《目标》中提出约束理论。 该理论强调找出限制系统产出的瓶颈环节，集中资源突破约束，从而提升整个系统的绩效。

五步聚焦法（TOC核心方法）
第1步：识别系统的约束（Identify）
找出限制系统产出的瓶颈环节
方法：流程分析、产能对比、数据统计
↓
第2步：决定如何充分利用约束（Exploit）
在不增加投资的前提下，挖掘约束环节的潜力
方法：减少浪费、消除停机、优化排程
↓
第3步：使其他一切服从于上述决定（Subordinate）
调整其他环节的节奏，配合约束环节
方法：设置缓冲、调整批量、协同作业
↓
第4步：打破（提升）系统的约束（Elevate）
通过投资或改进，提升约束环节的能力
方法：增加设备、技术改造、外协加工
↓
第5步：回到第1步，不要让惰性成为新的约束（Repeat）
旧约束打破后，新约束会出现，持续改进
警惕：不要让组织惰性阻碍持续改进
💡 轴承生产线TOC应用全流程

生产流程及产能（件/小时）：

下料(120) → 车削(80) → 热处理(100) → 磨削(90) → 装配(110)

第1步：识别约束

约束环节：车削（产能最低，80件/小时） 系统产出：受限于车削，整线产出只能是80件/小时 浪费分析： • 下料产能120，浪费40件/小时的能力（33%闲置） • 热处理产能100，浪费20件/小时（20%闲置） • 磨削产能90，浪费10件/小时（11%闲置） • 装配产能110，浪费30件/小时（27%闲置）

第2步：充分利用约束（零投资改善）

措施： ① 确保车削设备满负荷运转（减少停机、换模） ② 优先加工紧急订单（在车削环节优先排程） ③ 减少车削质量缺陷（避免浪费约束资源） ④ 优化车削工艺参数（在不影响质量前提下提速） 效果：车削产能从80提升到85件/小时（提升6.25%） 整线产出提升6.25%，零投资！

第3步：其他环节服从约束

措施： ① 在车削前建立缓冲库存（避免车削"饿死"） ② 下料环节不必全力生产，按车削节奏供料 ③ 后续环节确保及时处理车削产出（避免堆积） ④ 维修优先保障车削设备（预防性维护） ⑤ 质量控制重点前移到车削之前 效果：车削利用率从85%提升到95% 实际产出从80件/小时稳定提升至81件/小时

第4步：打破约束（需要投资）

方案 投资 效果 投资回报
方案A：增加1台车削设备 50万 产能翻倍至170 产能过剩
方案B：车削工序外协 0 成本增加15元/件 不经济
方案C：自动化改造 80万 产能提升至130 12个月
方案D：技术改造简化工序 20万 产能提升至110 6个月
选择方案D（投资回报最佳） 通过工艺优化，将车削工序从3道合并为2道 车削产能从81提升到110件/小时（提升36%！）

第5步：识别新约束（持续改进）

新的生产线产能： 下料(120) → 车削(110) → 热处理(100) → 磨削(90) → 装配(110) 新约束：磨削（产能最低，90件/小时） 系统产出：从80提升到90件/小时（提升12.5%） 继续应用五步聚焦法改善磨削环节... 通过持续改进，系统产出可能达到100件/小时（提升25%）
✓ TOC在成本管理中的应用价值：
  • 识别成本瓶颈：找出成本链条中的关键约束环节，集中资源突破
  • 优化资源配置：避免在非约束环节过度投入，提高投资效率
  • 系统思考：整体优化胜于局部优化，避免"救火"式管理
  • 持续改进：约束是动态的，需要持续识别和突破，形成改进循环
  • 快速见效：聚焦关键环节，短期内即可产生显著效果

3.3 质量成本管理

质量成本四大类别（PAF模型）
预防成本（P）

定义：为预防不合格品发生而支付的成本

包括：

  • 质量培训
  • 工艺评审
  • 供应商评审
  • 质量体系建设
  • 质量改进活动
鉴定成本（A）

定义：为评价产品是否合格而支付的成本

包括：

  • 来料检验
  • 工序检验
  • 最终检验
  • 检测设备维护
  • 检验人员工资
内部损失成本（IF）

定义：产品交付前因不合格产生的成本

包括：

  • 报废损失
  • 返工返修
  • 停机损失
  • 降级损失
  • 重新检验
外部损失成本（EF）

定义：产品交付后因不合格产生的成本

包括：

  • 客户退货
  • 质量索赔
  • 售后维修
  • 品牌损失
  • 法律诉讼
质量成本的经济规律
质量成本公式：
总质量成本 = 预防成本(P) + 鉴定成本(A) + 内部损失(IF) + 外部损失(EF)
质量成本的重要规律：
预防成本 ↑ → 损失成本 ↓↓（投入产出比极高）

经验数据（黄金法则）：

  • 预防成本增加1元 → 可减少损失成本10元
  • 工序检验成本增加1元 → 可减少最终检验和损失成本5元
  • 设计阶段发现缺陷 → 成本：1元
  • 生产阶段发现缺陷 → 成本：10元
  • 客户处发现缺陷 → 成本：100元
💡 质量成本优化实战案例

原质量成本结构（年产10万件，总质量成本200万元）：

类别 金额 占比 问题
预防成本 20万 10% 投入严重不足
鉴定成本 60万 30% 末端检验为主

外部损失 40万 20% 客诉多，品牌受损
合计 200万 100% 质量成本率：3.3%

改进措施：

策略1：加强源头控制 增加预防成本投入至50万元（+30万） • 加强工艺评审和FMEA分析 • 实施首件检验制度 • 加强操作培训和技能认证 • 改进设备精度和工装 策略2：实施SPC统计过程控制 增加鉴定成本至70万元（+10万） • 关键工序实施控制图 • 实时监控工序能力Cpk • 及时发现异常，快速响应 策略3：强化供应商质量 • 推行供应商质量改进计划 • 来料检验加严，不合格退货

合计 130万 100% ↓ 35%

效益分析：

增加投入： 预防成本增加：30万 鉴定成本增加：10万 合计投入：40万元 减少损失： 内部损失减少：72万 外部损失减少：38万 合计减少：110万元 净收益：110 - 40 = 70万元/年 投资回报率：70/40 = 175% 质量成本率：从3.3%降至2.2%（下降1.1%） 结论： 质量成本投入方向正确，向预防倾斜， 损失成本大幅下降，总质量成本显著降低， 同时产品质量提升，客户满意度提高！
