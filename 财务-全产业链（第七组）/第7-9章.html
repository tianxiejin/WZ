供应链管理理论

优化采购、库存和物流，降低供应链总成本

7.1 采购成本管理——TCO总拥有成本模型

核心概念
TCO(Total Cost of Ownership)总拥有成本模型突破了传统采购只关注"购买价格"的局限，从全生命周期视角评估采购成本。它强调采购决策不应仅看"买入价"，更要看"使用成本"和"处置成本"。
TCO = 购买成本 + 持有成本 + 使用成本 + 处置成本
TCO四大成本构成
1. 购买成本

包括:

  • 采购价格(单价×数量)
  • 运输费用
  • 关税、保险
  • 检验费

占比:通常占60-70%

2. 持有成本

包括:

  • 库存资金占用
  • 仓储费用
  • 保险费
  • 损耗、过期

占比:年库存周转成本约15-25%

3. 使用成本

包括:

  • 加工损耗
  • 不良品损失
  • 返工费用
  • 停机损失

占比:隐性成本，易被忽视

4. 处置成本

包括:

  • 报废处理费
  • 环保处理费
  • 残值回收

占比:通常占比较小

7.2 库存管理——经济订货批量EOQ

核心概念
经济订货批量(Economic Order Quantity, EOQ)模型通过平衡订货成本和库存持有成本，找出使总成本最低的最优订货批量。这是库存管理最经典的模型。
EOQ = √(2DS / H)

D = 年需求量
S = 单次订货成本
H = 单位年持有成本
EOQ模型的核心思想:
  • 订货成本与批量反比:订货次数越多，订货成本越高
  • 持有成本与批量正比:库存量越大，持有成本越高
  • 最优点:两种成本相等时，总成本最低
EOQ模型公式推导
年订货成本 = (年需求量/订货批量) × 单次订货成本
= (D/Q) × S
+
年持有成本 = (平均库存量) × 单位年持有成本
= (Q/2) × H
↓
总成本 TC = (D/Q)×S + (Q/2)×H
对Q求导，令 dTC/dQ = 0
-DS/Q² + H/2 = 0
Q² = 2DS/H
Q = √(2DS/H)
💡 EOQ模型应用:轴承钢采购优化

基础数据:

  • 年需求量D:1,200吨
  • 采购单价:6,000元/吨
  • 单次订货成本S:5,000元(包括采购人员工资分摊、物流协调、质检等)
  • 年持有成本率:20%(资金占用10%+仓储管理10%)
  • 单位年持有成本H = 6,000 × 20% = 1,200元/吨

计算EOQ:

EOQ = √(2 × 1200 × 5000 / 1200) = √(12,000,000 / 1200) = √10,000 = 100吨 最优订货策略: 每次订货:100吨 年订货次数 = 1200/100 = 12次 订货周期 = 12个月/12次 = 1个月 成本分析: 年订货成本 = (1200/100) × 5000 = 60,000元 年持有成本 = (100/2) × 1200 = 60,000元 年总成本 = 120,000元 (注:订货成本=持有成本，这是EOQ的特征)

与其他方案对比:

订货批量 订货次数 订货成本 持有成本 总成本
50吨 24次 120,000元 30,000元 150,000元
100吨(EOQ) 12次 60,000元 60,000元 120,000元
200吨 6次 30,000元 120,000元 150,000元
300吨 4次 20,000元 180,000元 200,000元
EOQ效益分析: vs 小批量(50吨):节约30,000元/年(20%) vs 大批量(200吨):节约30,000元/年(20%) 实施建议: 1. 与供应商协商，建立月度供货协议 2. 设置安全库存15吨(考虑1周交货期+波动) 3. 设置再订货点:月消耗100吨+安全库存15吨=115吨 4. 当库存降至115吨时，自动发起采购 5. 定期(每季度)重新评估EOQ参数 潜在风险与应对: • 价格波动风险 → 与供应商签订框架协议锁定价格 • 需求波动风险 → 设置合理安全库存 • 供应中断风险 → 培养备选供应商

7.3 JIT准时化生产

核心思想
"只在需要的时候，按需要的量，生产所需的产品"
Just-In-Time(JIT)——丰田生产方式的核心
JIT三大支柱
1. 准时化(JIT)

目标:零库存，按需生产

方法:

  • 拉动式生产
  • 看板管理
  • 均衡生产
  • 小批量流动
2. 自働化(Jidoka)

目标:质量内建，异常停止

方法:

  • 赋予机器判断力
  • 异常自动停机
  • 源流管理
  • 防错装置
3. 持续改善(Kaizen)

目标:消除浪费，追求完美

方法:

  • 全员参与改善
  • PDCA循环
  • 标准化作业
  • 可视化管理
拉动式生产vs推动式生产
对比维度 推动式生产(传统) 拉动式生产(JIT)
生产驱动 计划驱动，提前排产 需求驱动，按需生产
信息流向 上游推给下游 下游向上游要货
库存水平 高库存(在制品多) 低库存(理想为零)
生产批量 大批量(减少换模) 小批量(单件流)
问题暴露 库存掩盖问题 问题立即暴露
应对变化 反应慢，库存积压 快速响应，灵活调整
质量控制 事后检验，返工多 过程控制，源流管理

8.1 统计过程控制(SPC)

核心概念
统计过程控制(Statistical Process Control, SPC)是利用统计方法对过程进行监控，区分过程中的正常波动(普通原因)和异常波动(特殊原因)，及时发现异常并采取纠正措施，使过程保持在稳定可控状态。
SPC的核心理念:
  • 预防为主:通过过程控制预防不合格品产生，而非事后检验
  • 数据说话:用数据和统计方法识别问题，而非经验判断
  • 持续改进:不断缩小过程波动，提高过程能力
控制图原理
控制图 = 中心线(CL) + 上控制限(UCL) + 下控制限(LCL)
上控制限 UCL = μ + 3σ
99.73%的点应该在控制限内
↓
中心线 CL = μ
过程均值(目标值)
↓
下控制限 LCL = μ - 3σ
控制限由3σ原则确定
控制图判异准则(8大判异规则)
规则 判异标准 示意 可能原因
规则1 有点超出控制限 点在UCL或LCL外 设备故障、材料问题、人为失误
规则2 连续9点在中心线同侧 趋势偏移 工具磨损、原料变化
规则3 连续6点递增或递减 趋势存在 刀具磨损、温度漂移
规则4 连续14点交替上下 过度调整 操作员频繁调整
规则5 连续3点中有2点落在2σ外 波动增大 过程不稳定
规则6 连续5点中有4点落在1σ外 波动偏大 过程变化
规则7 连续15点在1σ内 波动过小 数据造假、分组不当
规则8 连续8点在1σ外 波动过大 混合了多个分布
过程能力指数Cpk
过程能力指数反映过程满足规格要求的能力
Cp = (USL - LSL) / 6σ
Cpk = min[(USL - μ)/3σ, (μ - LSL)/3σ]

USL = 规格上限
LSL = 规格下限
μ = 过程均值
σ = 过程标准差

Cpk评价标准:

  • Cpk ≥ 1.67:优秀(六西格玛水平)
  • 1.33 ≤ Cpk < 1.67:良好(能力充分)
  • 1.00 ≤ Cpk < 1.33:尚可(能力不足，需改进)
  • Cpk < 1.00:不合格(能力严重不足)
💡 SPC应用案例:磨削工序控制

工序信息:

  • 工序:磨削外圈外径
  • 规格要求:Φ100±0.05mm (USL=100.05, LSL=99.95)
  • 取样:每小时抽取5件，测量尺寸

数据收集(某班次20组数据):

样本均值 X̄ = 100.00mm 样本标准差 σ = 0.012mm 计算过程能力: Cp = (100.05 - 99.95) / (6 × 0.012) = 0.10 / 0.072 = 1.39 Cpk = min[(100.05-100.00)/(3×0.012), (100.00-99.95)/(3×0.012)] = min[0.05/0.036, 0.05/0.036] = min[1.39, 1.39] = 1.39 结论:Cpk=1.39，处于"良好"水平，能力充分

控制图监控(发现异常):

控制限计算: UCL = 100.00 + 3×0.012 = 100.036mm CL = 100.00mm LCL = 100.00 - 3×0.012 = 99.964mm 异常发现(第18组): 第18组样本:100.042, 100.045, 100.050, 100.048, 100.052 组均值:100.047mm > UCL(100.036mm) → 触发规则1:有点超出控制限! 立即行动: 1. 停机检查 2. 发现砂轮钝化 3. 更换砂轮 4. 重新首件检验 5. 恢复生产

效果分析:

实施SPC前(事后检验): 不良品率:1.5% 年产量:10万件 年不良品:1,500件 单件材料成本:50元 年质量损失:1,500 × 50 = 75,000元 实施SPC后(过程控制): 不良品率:0.3% 年产量:10万件 年不良品:300件 年质量损失:300 × 50 = 15,000元 年节约质量成本:60,000元(80%) 附加效益: • 及时发现异常,避免批量不良 • 质量稳定,客户信任度提升 • 减少返工和报废 • 生产效率提高(减少停机检查) 投入: • SPC软件:2万元 • 培训:1万元 • 测量设备校准:0.5万元 • 合计:3.5万元 投资回报率:60,000/35,000 = 171%，回收期:7个月

8.2 六西格玛管理

核心概念
"3.4 PPM(百万分之3.4的缺陷率)"
Six Sigma——追求接近完美的质量管理方法
六西格玛(Six Sigma, 6σ)是一种追求完美质量的管理方法，目标是将过程质量控制在6个标准差水平，即不良率低于3.4PPM(百万分之3.4)。它通过DMAIC或DMADV结构化方法，系统地消除缺陷和浪费。
西格玛水平对照表
西格玛水平 不良率(PPM) 良品率 评价 典型案例
2σ 308,537 69.15% 非常差 落后企业
3σ 66,807 93.32% 一般 中国制造业平均
4σ 6,210 99.38% 良好 行业领先企业
5σ 233 99.977% 优秀 丰田等标杆
6σ 3.4 99.9997% 世界级 摩托罗拉、通用电气
DMAIC五步改进法(针对现有流程)
D - Define(定义阶段)
• 识别改进项目
• 定义问题和目标
• 组建项目团队
工具:项目章程、SIPOC图、VOC(客户之声)
↓
M - Measure(测量阶段)
• 确定关键质量特性CTQ
• 建立测量系统
• 收集基线数据
工具:MSA测量系统分析、过程能力分析、数据收集计划
↓
A - Analyze(分析阶段)
• 识别根本原因
• 验证因果关系
• 确定关键因子
工具:鱼骨图、5Why、假设检验、方差分析、回归分析
↓
I - Improve(改进阶段)
• 提出改进方案
• 试验验证
• 实施优化
工具:DOE实验设计、头脑风暴、FMEA、Poka-Yoke防错
↓
C - Control(控制阶段)
• 标准化改进成果
• 建立控制系统
• 持续监控
工具:控制图、作业指导书、控制计划、审核机制
💡 六西格玛项目案例:降低轴承噪音不合格率

D-定义阶段:

问题:某型号轴承噪音不合格率3.5%(3σ水平)，客户抱怨多 目标:6个月内将噪音不合格率降至0.5%(4.5σ水平) 项目效益:年产10万件，单件利润40元，合格率提升可增加利润12万元/年

M-测量阶段:

CTQ(关键质量特性):轴承振动噪音值 规格要求:≤60dB 测量系统:振动噪音测试仪(经MSA分析合格) 数据收集:连续30天，每天抽检20件 基线数据: 样本量:600件 不合格:21件 不合格率:3.5% 当前西格玛水平:3.0σ

A-分析阶段:

潜在原因分析(鱼骨图): • 材料:钢材纯净度、硬度不均 • 机器:磨床精度、主轴跳动 • 方法:磨削参数、装配工艺 • 人员:操作技能、检验标准 • 环境:温度波动、振动干扰 数据分析(相关性检验): 经过假设检验和方差分析，发现: 1. 磨削砂轮粒度不当(P值<0.05，显著相关) 2. 滚道表面粗糙度过大(P值<0.01，高度相关) 3. 装配间隙控制不稳定(P值<0.05，显著相关) 根本原因:磨削工艺参数设置不当 + 装配间隙控制不严

I-改进阶段:

改进方案: 1. 优化磨削工艺: • 调整砂轮粒度从80#改为120# • 优化切削参数(进给、转速) • 严格砂轮修整周期 2. 改进装配工艺: • 引入自动测量装配间隙 • 制定严格的间隙控制标准(0.02-0.03mm) • 100%检测装配间隙 3. 过程控制: • 关键工序实施SPC控制 • 首件三件检验制度 • 每班点检设备精度 DOE实验验证: 设计2³因子实验，验证最优参数组合 实验结果:噪音不合格率降至0.4%

C-控制阶段:

标准化: • 更新工艺文件和作业指导书 • 培训所有操作人员和检验员 • 建立磨削和装配工序SPC控制图 持续监控: • 每班记录噪音检测数据 • 每周分析SPC控制图 • 每月评审不合格趋势 审核机制: • 质量部每月审核工艺执行情况 • 工艺部每季度评估改进效果

第九章

战略管理理论

将成本管理与企业战略相结合，构建持久竞争优势

9.1 波特竞争战略

核心概念
迈克尔·波特(Michael Porter)提出企业要获得竞争优势，必须在战略定位上作出明确选择:成本领先、差异化或集中化。不同的战略定位决定了企业不同的成本管理重点和方法。
波特三大基本竞争战略
1. 成本领先战略

核心:以低于竞争对手的成本提供产品或服务

适用:标准化产品、大规模生产、价格敏感市场

关键要素:

  • 规模经济
  • 学习曲线效应
  • 严格的成本控制
  • 高效率运营
  • 标准化生产

典型案例:沃尔玛、格兰仕

2. 差异化战略

核心:提供独特价值，获得溢价能力

适用:品牌敏感、质量要求高、定制化需求

关键要素:

  • 产品创新
  • 品牌建设
  • 卓越质量
  • 优质服务
  • 技术领先

典型案例:苹果、宝马

3. 集中化战略

核心:聚焦特定细分市场

适用:资源有限、细分市场有独特需求

关键要素:

  • 市场聚焦
  • 专业化
  • 深度服务
  • 快速响应

两种形式:
• 集中成本领先
• 集中差异化

成本领先战略深度解析(轴承行业重点)
成本领先战略的实施路径:
规模效应
通过大批量生产分摊固定成本
轴承:年产百万级，单位固定成本大幅降低
+
学习曲线
累计产量增加，单位成本下降
轴承:工艺成熟后，效率提升20-30%
+
工艺优化
持续改进生产工艺，降低单位成本
轴承:减少工序、提高自动化、优化参数
+
供应链整合
优化采购、降低库存、提升周转
轴承:与供应商战略合作，JIT供货
↓
成本优势 → 价格优势 → 市场份额 → 规模效应 → 成本进一步降低
形成正向飞轮效应!
⚠️ 成本领先战略的风险:
  • 过度降本伤害质量:一味压缩成本可能损害产品质量，失去客户信任
  • 技术变革颠覆:新技术可能让既有成本优势失效
  • 竞争对手模仿:其他企业也可能实现低成本，优势消失
  • 忽视差异化:产品同质化严重，陷入价格战泥潭
  • 灵活性下降:为了效率而牺牲灵活性，难以快速响应市场变化
瓦轴的战略选择:成本领先+品质保证
瓦轴集团的差异化成本领先战略:
战略定位: "在保证质量的前提下，实现成本领先" 既要成本优势，又要质量保证——不做低价低质产品 实施策略: 1. 规模优势(成本领先基础) • 年产能数亿套，中国最大轴承制造商之一 • 规模分摊固定成本，单位成本显著降低 • 采购量大，议价能力强 2. 工艺优化(降本不降质) • 持续工艺改进和技术创新 • 提高自动化水平，降低人工成本 • 优化工序，在不影响质量前提下提效 3. 质量体系(品质保证) • 严格的质量管理体系(ISO/TS 16949) • 关键工序100%检验 • 持续质量改进(六西格玛、SPC) 4. 供应链整合(全链降本) • 与核心供应商战略合作 • 实施JIT和VMI，降低库存 • 全产业链成本联动模型 5. 精益生产(消除浪费) • 全面推行精益生产 • 识别和消除七大浪费 • 提升OEE和设备利用率 战略效果: 成本:低于国际品牌20-30%，与国内品牌相当或更低 质量:达到或接近国际品牌水平 价格:介于国际品牌和低端品牌之间 定位:高性价比的优质产品 竞争优势: • 对比国际品牌:价格更优，性价比高 • 对比国内低端品牌:质量更好,品牌更强 • 适合中国市场:兼顾质量与价格

9.2 平衡计分卡(BSC)

核心概念
平衡计分卡(Balanced Scorecard, BSC)由哈佛商学院罗伯特·卡普兰和大卫·诺顿提出，是一种战略绩效管理工具。它从财务、客户、内部流程、学习与成长四个维度全面衡量企业绩效，将战略目标转化为可执行的行动计划。
BSC四大维度
1. 财务维度

核心问题:我们如何满足股东?

典型指标:

  • 营业收入增长率
  • 净利润率
  • 投资回报率ROI
  • 成本降低率
  • 资产周转率

降本指标:单位成本、成本费用率、料工费占比

2. 客户维度

核心问题:客户如何看待我们?

典型指标:

  • 客户满意度
  • 市场份额
  • 客户保留率
  • 准时交付率
  • 产品质量合格率

降本关联:高质低价产品提升客户满意度

3. 内部流程维度

核心问题:我们必须擅长什么?

典型指标:

  • 生产效率OEE
  • 不良品率
  • 生产周期
  • 库存周转率
  • 工艺合格率

降本核心:工艺优化、效率提升、质量改进

4. 学习与成长维度

核心问题:我们如何持续改进?

典型指标:

  • 员工满意度
  • 培训完成率
  • 技术创新数量
  • 改善提案数
  • 知识管理系统

降本基础:员工能力提升、创新文化培育
