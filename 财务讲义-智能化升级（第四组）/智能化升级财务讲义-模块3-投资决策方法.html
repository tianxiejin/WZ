一、资金时间价值：为什么今天的100万≠明年的100万？

在学习NPV和IRR之前，必须先理解一个重要概念：资金时间价值（Time Value of Money）。

1.1 什么是资金时间价值？

简单说：今天的钱比未来的钱更值钱。

【思考题】现在给你100万 vs. 5年后给你100万，你选哪个？

大多数人会选择"现在"，为什么？

  1. 机会成本：现在拿到100万，可以存银行拿利息，或者投资赚收益
  2. 通货膨胀：5年后的100万购买力下降，可能只相当于现在的80-90万
  3. 风险：5年后能否真的拿到100万，存在不确定性

结论：未来的钱要折现（Discount）到现在，才能公平比较。

1.2 现值（PV）与未来值（FV）

资金时间价值核心公式
FV = PV × (1 + r)ⁿ

公式说明：
FV（Future Value）= 未来值
PV（Present Value）= 现值
r = 折现率（discount rate），也称贴现率
n = 年数

这个公式可以从两个方向使用：
1) 已知现值求未来值：未来值 = 现值 × (1+r)ⁿ（复利计算）
2) 已知未来值求现值：现值 = 未来值 / (1+r)ⁿ（折现计算）

【案例1】现值与未来值计算

问题1：现在投资100万，年化收益率10%，5年后值多少钱？

计算过程

FV = 100万 × (1 + 10%)⁵
= 100万 × 1.6105
= 161.05万

问题2：5年后要拿到100万，按10%折现率，现在应该投资多少？

计算过程

PV = 100万 / (1 + 10%)⁵
= 100万 / 1.6105
= 62.09万

理解：5年后的100万，按10%折现，只相当于现在的62.09万！这就是资金时间价值的威力。

1.3 折现率（r）如何确定？

折现率是NPV计算的核心参数，代表企业的资金成本。

折现率的三种确定方法

  1. 使用加权平均资本成本（WACC）
    • WACC = 权益资本成本 × 权益比例 + 债务资本成本 × 债务比例 × (1-税率)
    • 大中型企业通常采用WACC，典型值8-12%
  2. 使用银行贷款利率
    • 如果项目主要靠贷款，可以用贷款利率（如5-6%）
    • 要加上风险溢价，一般在贷款利率基础上+2-4%
  3. 使用机会成本
    • 如果放弃的是其他投资机会，用那个项目的收益率
    • 例如：放弃了收益率12%的理财产品，折现率至少要12%

实务建议：瓦轴作为国有企业，建议使用WACC法。如果WACC不清楚，可以用10%作为默认折现率（这是制造业的典型水平）。

二、净现值（NPV）法：最重要的投资决策工具

2.1 NPV的概念

净现值（Net Present Value）= 项目所有未来现金流的现值 - 初始投资

净现值（NPV）计算公式
NPV = -I₀ + Σ [CFₜ / (1+r)ᵗ]

公式说明：
I₀ = 初始投资（第0年，用负号表示现金流出）
CFₜ = 第t年的净现金流（现金流入-现金流出）
r = 折现率
t = 年份（1, 2, 3, ..., n）
Σ = 求和符号，表示把每年的现金流现值加总

白话翻译：NPV = 把未来每年赚的钱折现到现在，再减去初始投资

NPV决策规则

  • NPV > 0：项目可行，能创造价值，应该投资
  • NPV = 0：项目盈亏平衡，刚好达到要求收益率，可投可不投
  • NPV < 0：项目不可行，会损毁价值，不应该投资
  • 多方案选择：NPV越大越好，选NPV最大的方案

2.2 NPV完整计算案例

【核心案例】某轴承生产线智能化改造NPV分析

项目背景：

  • 初始投资：1260万元（第0年）
  • 项目寿命：10年
  • 年净现金流：952万元/年（从模块2的效益测算）
  • 折现率：10%（WACC）
  • 残值：设备10年后残值100万元

NPV计算步骤

步骤1：列出现金流量表

0 1260 0 -1260 1.0000 -1260.00
1 0 952 952 0.9091 865.45
2 0 952 952 0.8264 786.77
3 0 952 952 0.7513 715.24
4 0 952 952 0.6830 650.22
5 0 952 952 0.6209 591.11

步骤2：理解折现系数

例如第1年：1 / (1+10%)¹ = 0.9091

经济意义：投资这个项目，在满足10%资本成本的前提下，还能额外创造4628万元的价值。

2.3 NPV的优点与局限

优点 局限
✓ 考虑了资金时间价值 ✗ 对折现率敏感，折现率变化NPV变化大
✓ 考虑了项目全生命周期现金流 ✗ 难以比较不同规模项目（绝对数指标）
✓ 直接反映项目价值创造能力 ✗ 计算相对复杂，需要Excel辅助
✓ 多方案选择时，选NPV最大的 ✗ 假设现金流可以按r再投资（理想化）

三、内部收益率（IRR）法：项目的真实回报率

3.1 IRR的概念

内部收益率（Internal Rate of Return）= 使得NPV=0的折现率，也就是项目的真实回报率。

IRR的定义
NPV = -I₀ + Σ [CFₜ / (1+IRR)ᵗ] = 0
求解IRR：找到一个折现率，使得NPV刚好等于0

白话理解：IRR就是项目能够承受的最高资金成本。如果你的资金成本低于IRR，项目就赚钱；如果高于IRR，项目就亏钱。

IRR决策规则

  • IRR > r（资金成本）：项目可行，应该投资
  • IRR = r：项目盈亏平衡，可投可不投
  • IRR < r：项目不可行，不应该投资
  • 多方案选择：IRR越高越好，但要结合NPV综合判断

3.2 IRR的计算方法

IRR没有直接公式，需要用试错法或Excel函数求解。

【案例2】IRR计算方法演示

项目数据（沿用上面的案例）：

  • 初始投资：-1260万
  • 年现金流：952万/年（1-9年），1052万（第10年）

方法1：试错法（手工计算）

尝试的折现率 计算的NPV 结论
10% 4628万 折现率太低，NPV>0
50% 1523万 还是太低
70% 258万 接近了
75% 54万 很接近
76.5% ≈ 0 这就是IRR！

方法2：Excel函数（推荐）

步骤1：在Excel中列出现金流 
A列：年份    B列：现金流
0           -1260
1            952
2            952
...
10           1052
步骤2：使用IRR函数 
=IRR(B1:B11)
结果：76.5%

决策结论

IRR = 76.5% > r = 10%，该项目应该投资！

经济意义：这个项目的真实回报率高达76.5%，远超资金成本10%，是个非常优秀的投资项目！

3.3 NPV与IRR的关系

NPV与IRR的关系

  • 当折现率 < IRR时，NPV > 0，项目可行
  • 当折现率 = IRR时，NPV = 0，项目盈亏平衡
  • 当折现率 > IRR时，NPV < 0，项目不可行

记忆技巧：IRR是NPV=0时的那个特殊折现率。

3.4 IRR的局限性

IRR的缺陷

  1. 多重IRR问题：如果现金流正负交替（先流入后流出再流入），可能有多个IRR，无法判断
    • 例如：矿山开采项目，最后要支付环境恢复费用
  2. 规模差异问题：IRR相同，但投资规模不同，选哪个？
    • 项目A：投资100万，IRR 30%，NPV 50万
    • 项目B：投资1000万，IRR 25%，NPV 800万
    • 应该选B（NPV更大），但IRR会误导选A
  3. 再投资假设不现实：IRR假设未来现金流可以按IRR再投资，这往往不现实

实务建议：以NPV为主，IRR为辅。NPV是绝对价值指标，更可靠；IRR是相对指标，便于理解和沟通。当NPV和IRR结论冲突时，以NPV为准。

四、投资回收期法：多久能回本？

4.1 静态回收期

静态回收期（Payback Period）= 用累计净现金流收回初始投资所需的时间。

静态回收期计算方法
静态回收期 = 累计净现金流刚好转正的年份
更精确的公式：回收期 = 最后一个负累计现金流年份 + |该年累计现金流| / 下一年净现金流

【案例3】静态回收期计算

年份 净现金流（万元） 累计净现金流（万元）
0 -1260 -1260
1 952 -308
2 952 +644
3 952 1596
... ... ...

精确计算

第1年末累计：-308万
第2年末累计：+644万
回收期 = 1 + 308/952 = 1 + 0.32 = 1.32年

结论：该项目1.32年就能回本，回收期很短。

4.2 动态回收期（折现回收期）

动态回收期考虑了资金时间价值，用折现后的累计净现金流计算回收期，更准确。

【案例4】动态回收期计算

年份 净现金流 折现系数(10%) 现值 累计现值
0 -1260 1.0000 -1260 -1260
1 952 0.9091 865.45 -394.55
2 952 0.8264 786.77 +392.22
... ... ... ... ...

动态回收期计算

第1年末累计现值：-394.55万
第2年末累计现值：+392.22万
动态回收期 = 1 + 394.55/786.77 = 1 + 0.50 = 1.50年

对比：

  • 静态回收期：1.32年
  • 动态回收期：1.50年（更保守，更可靠）

4.3 回收期法的优缺点

优点 缺点
✓ 直观易懂，管理层容易接受 ✗ 忽略回收期后的现金流
✓ 强调资金快速回笼，降低风险 ✗ 静态回收期不考虑时间价值
✓ 计算简单 ✗ 不能反映项目盈利能力

使用建议：回收期法作为辅助指标，与NPV、IRR配合使用。对于风险较高的项目，可以设定最长回收期限（如3年），作为筛选条件。

六、实务中的注意事项

现金流预测要谨慎

  1. 区分会计利润与现金流
    • NPV用的是现金流，不是利润
    • 折旧不是现金流出，不能扣除（但折旧抵税要考虑）
  2. 考虑营运资本变化
    • 产能提升可能需要增加库存（现金流出）
    • 项目结束时营运资本回收（现金流入）
  3. 效益爬坡期
    • 第1年可能只有预期效益的50-70%
    • 第2年80-90%，第3年才达到100%
  4. 保守估计
    • 效益宁可低估，成本宁可高估
    • 留有安全边际

七、本模块小结

核心要点回顾

  1. 资金时间价值：今天的钱比未来的钱更值钱，要用折现率换算
  2. NPV是核心：NPV > 0就投，NPV < 0就不投，多方案选NPV最大的
  3. IRR是补充：IRR是项目真实回报率，IRR > 资金成本就投
  4. 回收期看风险：回收期越短风险越低，通常作为筛选条件
  5. 决策优先级：NPV（主） > IRR（辅） > 回收期（补充）
