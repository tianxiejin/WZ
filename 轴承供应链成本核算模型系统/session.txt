轴承供应链总成本主要包括以下几个维度：
 直接成本
- 原材料成本 ：钢材、滚珠、保持架、密封圈等
- 直接人工成本 ：生产工人工资、奖金、津贴等

基于作业成本法，我们将间接成本分解为以下作业：

- 采购作业 ：供应商开发、询价比价、合同管理等
- 质检作业 ：进料检验、过程检验、成品检验等
- 仓储作业 ：入库作业、存储保管、出库作业等
- 运输作业 ：内部运输、外部配送等
- 订单作业 ：订单处理、排程协调等 资金成本
- 库存资金占用成本 ：原材料、在制品、产成品的资金成本

- 质量损失成本 ：退货、索赔、品牌影响等
- 供应中断成本 ：缺货损失、紧急采购溢价等
- 价格波动成本 ：原材料价格波动风险
## 二、模型核心算法
### 总成本计算公式
```
总成本 = 直接材料成本 + 直接人工成本 + 直接制造费用 + 作业成本 + 资金成本 + 风险成本
```
### 作业成本计算
```
作业成本 = ∑(作业成本池金额 × 作业动因消耗量 ÷ 作业动因总量)
```
### 资金占用成本计算
```
资金占用成本 = 平均库存金额 × 资金成本率 × 库存周转天数 ÷ 365
```
