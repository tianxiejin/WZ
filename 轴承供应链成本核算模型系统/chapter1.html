第一章：财务基础理论

对于供应链管理人员来说，理解成本的基本概念是进行有效成本控制的前提。本章将介绍供应链管理中必需的财务基础知识。

1.1 成本的基本概念

成本的定义：成本是指企业为取得资产或达到特定目的而实际发生或应当发生的价值牺牲。在供应链管理中，成本不仅包括采购价格，还涉及运输、仓储、管理等多个方面。

成本的分类

分类标准 类型 说明
按经济用途 生产成本 直接用于产品生产的费用，如原材料、直接人工等
期间费用 不能直接归属于特定产品的费用，如管理费用、销售费用
按计入产品成本的方式 直接成本 可以直接计入特定产品成本的费用，如直接材料
间接成本 不能直接计入特定产品成本的费用，需按一定标准分配，如制造费用
按与业务量的关系 固定成本 在一定时期内不随业务量变化的成本，如租金、折旧
变动成本 随业务量成正比例变化的成本，如直接材料、计件工资
混合成本 同时具有固定和变动成分的成本，如水电费

1.2 成本核算基础

成本核算是成本管理的基础工作，其目的是正确计算产品成本，为经营决策提供依据。

成本核算的原则

  • 实际成本原则：各项财产物资应当按取得时的实际成本计价
  • 一致性原则：成本核算所采用的方法应保持一致
  • 配比原则：收入与其相关的成本费用应当相互配比
  • 权责发生制原则：凡当期已经实现的收入和已经发生或应当负担的费用，不论款项是否收付，都应当作为当期的收入和费用

成本核算的基本要求

  1. 划清收益性支出与资本性支出的界限
  2. 划清产品制造成本与期间费用的界限
  3. 划清各种产品成本的界限
  4. 划清本期完工产品与期末在产品成本的界限

实例解析

某轴承制造企业采购一批钢材，采购价格为100万元，运输费用5万元，入库前挑选整理费用1万元。则这批钢材的采购成本为：

采购成本 = 采购价格 + 运输费用 + 入库前挑选整理费用
= 100万 + 5万 + 1万 = 106万元

1.3 财务报表基础

了解基本的财务报表有助于供应链管理人员更好地理解成本数据在企业整体财务中的位置。

主要财务报表

  • 资产负债表：反映企业在某一特定日期的财务状况，显示企业的资产、负债和所有者权益
  • 利润表：反映企业在一定会计期间的经营成果，显示收入、费用和利润
  • 现金流量表：反映企业在一定会计期间现金和现金等价物流入和流出情况
重要概念：供应链成本在财务报表中通常体现在存货项目中，包括原材料、在产品和产成品。采购价格差异、运输费用等会影响存货的账面价值。
