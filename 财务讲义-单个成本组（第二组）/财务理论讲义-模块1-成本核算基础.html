模块 1

成本核算基础

从零开始理解成本——制造业成本核算的基本逻辑

一、什么是成本?

📖 核心概念：成本

成本是企业为了生产产品、提供服务而消耗的各种资源的货币表现。

通俗地说：做一件产品,你花了多少钱,这就是成本。

1.1 用生活案例理解成本

🏠 案例1：做一顿饭的成本

假设你在家做一顿饭,需要:

  • 食材：买菜、肉、油、盐等 → 这相当于直接材料
  • 人工：你花了2小时做饭 → 这相当于直接人工(如果按小时计算你的时间价值)
  • 其他：用电、用气、厨具折旧 → 这相当于制造费用

这顿饭的成本 = 食材费 + 人工费 + 水电气费

制造轴承也是一样的道理!

⚙️ 案例2：生产一套轴承6206的成本

  • 钢材、润滑脂 → 直接材料成本
  • 工人工资(车工、磨工等) → 直接人工成本
  • 设备折旧、电费、车间管理费 → 制造费用

这套轴承的成本 = 材料成本 + 人工成本 + 制造费用

1.2 成本与费用的区别

成本 (Cost)

  • 与生产直接相关
  • 可以归属到具体产品
  • 计入产品价值
  • 例如：原材料、生产工人工资

费用 (Expense)

  • 与经营活动相关
  • 难以归属到具体产品
  • 直接计入当期损益
  • 例如：销售费用、管理费用
记忆口诀：生产产品花的钱叫成本，卖产品、管公司花的钱叫费用。本课题重点关注产品成本。

二、成本的分类

成本有很多种分类方法,不同的分类服务于不同的管理目的。我们重点掌握以下三种分类:

2.1 按经济性质分类（最基础）

成本要素 定义 轴承制造举例
直接材料 构成产品实体的原材料及主要材料 GCr15轴承钢、润滑脂、防锈纸、包装材料
直接人工 直接参与产品生产的工人工资 锻工、车工、磨工、装配工的工资及社保
制造费用 生产产品发生的间接费用 设备折旧、电费、车间管理人员工资、维修费
产品成本基本公式
产品总成本 = 直接材料成本 + 直接人工成本 + 制造费用
(俗称"料工费"三要素)

2.2 按成本与产品的关系分类

分类 定义 举例
直接成本 能直接计入某种产品成本的费用 生产6206型轴承的钢材、专用工人工资
间接成本 不能直接计入,需要分配的费用 车间主任工资、电费（多种产品共用）
判断技巧：问自己"这笔费用能不能明确地说是为某个产品花的?"能明确说清楚的就是直接成本,说不清楚需要按一定方法分摊的就是间接成本。

2.3 按成本与产量的关系分类（重要）

分类 特点 举例 管理意义
变动成本 随产量变化而成正比例变化 原材料、计件工资 生产越多,成本越高
固定成本 在一定范围内不随产量变化 设备折旧、管理人员工资 产量增加可摊薄单位固定成本
混合成本 既有固定部分,也有变动部分 电费（基本电费+用电量电费） 需要拆分分析

💡 理解固定成本与变动成本的意义

场景：车间设备折旧费每月10万元（固定成本）

  • 生产1万套轴承 → 单位折旧成本 = 10万 ÷ 1万 = 10元/套
  • 生产2万套轴承 → 单位折旧成本 = 10万 ÷ 2万 = 5元/套

启示：提高产能利用率,可以降低单位固定成本,这就是规模效应!

三、成本核算的基本概念

3.1 成本核算对象

📖 什么是成本核算对象?

成本核算对象就是你要计算哪个"东西"的成本。

可以是：一件产品、一批产品、一道工序、一个车间、一个项目……

在我们的课题中,成本核算对象 = 单个产品型号(例如:6206型轴承)

3.2 成本项目

📖 什么是成本项目?

成本项目就是把成本按性质分成几大类,便于管理和分析。

制造业常用的成本项目分类:

一级成本项目 二级成本项目(细分) 轴承制造举例
直接材料 主要材料 GCr15轴承钢
辅助材料 润滑脂、防锈纸
外购件 标准件、密封圈
直接人工 计件工资 按件数支付的加工费
计时工资 按工时支付的工资
制造费用 设备折旧 车床、磨床折旧
能源费用 电费、水费、气费
维修费 设备维护、保养
间接人工 车间主任、质检员工资
其他 低值易耗品、办公费

3.3 成本核算方法

根据生产特点不同,选择不同的成本核算方法:

核算方法 适用场景 核算特点 轴承适用
品种法 大批量、单步骤生产 以产品品种为对象归集成本 部分适用
分批法 单件、小批量生产 以批次为对象归集成本 定制产品
分步法 大批量、多步骤生产 按生产步骤归集成本,逐步结转 ★主要方法

⚙️ 轴承生产为什么适合分步法?

轴承生产有明显的加工步骤:

  1. 锻造 → 棒料锻造成型
  2. 车削 → 粗车、精车外圆和内孔
  3. 热处理 → 淬火、回火
  4. 磨削 → 精磨内外圆、端面
  5. 装配 → 组装内外圈、滚珠、保持架

每一步都发生成本,分步法可以计算每一步的成本,便于控制和分析。

四、产品成本计算的基本流程

4.1 成本核算流程图

制造业产品成本核算流程

第一步：确定成本核算对象
   ↓ (例如：6206型轴承)

第二步：设置成本项目
   ↓ (直接材料、直接人工、制造费用)

第三步：归集生产费用
   ├─ 直接费用 → 直接计入产品成本
   └─ 间接费用 → 需要分配

第四步：分配间接费用
   ↓ (按工时、机器工时、产量等分配)

第五步：计算产品成本
   ├─ 在产品成本 (还未完工的)
   └─ 完工产品成本 (已完工的)

第六步：计算单位成本
   完工产品总成本 ÷ 完工产量 = 单位产品成本

4.2 成本核算的关键点

🎯 成本核算四大关键

  • 准确归集：各项费用要正确归集到对应的成本对象
  • 合理分配：间接费用要选择合理的分配方法(这是本课题的难点!)
  • 及时结转：完工产品与在产品要及时区分
  • 正确计算：单位成本计算公式要正确

4.3 一个完整的成本计算案例

📊 案例：计算6206轴承的成本（简化版）

背景：某月生产6206轴承10,000套,全部完工。

第一步：归集直接材料成本

  • 领用GCr15钢材：4,000 kg × 12.5元/kg = 50,000元
  • 领用润滑脂：50 kg × 80元/kg = 4,000元
  • 领用包装材料：3,000元
  • 直接材料合计：57,000元

第二步：归集直接人工成本

  • 生产工人工资：28,000元
  • 社保公积金：7,000元
  • 直接人工合计：35,000元

第三步：分配制造费用

  • 本月车间制造费用总额：120,000元
  • 本月车间总工时：6,000小时
  • 分配率 = 120,000 ÷ 6,000 = 20元/小时
  • 6206产品工时：2,000小时
  • 分配到6206的制造费用 = 2,000 × 20 = 40,000元

第四步：计算总成本

6206轴承总成本
直接材料：57,000元
直接人工：35,000元
制造费用：40,000元
─────────────────
总成本：132,000元

第五步：计算单位成本

单位成本
单位成本 = 132,000元 ÷ 10,000套
= 13.2元/套

第六步：成本构成分析

成本项目 金额(元) 占比
直接材料 57,000 43.2%
直接人工 35,000 26.5%
制造费用 40,000 30.3%
合计 132,000 100%

五、本模块要点总结

📚 核心知识点回顾

  • 成本定义：生产产品消耗的资源的货币表现
  • 成本三要素：直接材料 + 直接人工 + 制造费用 (料工费)
  • 直接/间接：能直接归属就是直接成本,需要分配就是间接成本
  • 变动/固定：随产量变化是变动成本,不随产量变化是固定成本
  • 核算对象：我们要算哪个产品的成本(本课题是单个产品型号)
  • 核算方法：轴承制造主要用分步法(按工序逐步核算)
  • 核算流程：确定对象→设置项目→归集费用→分配费用→计算成本
学以致用：

回到工作岗位,观察一下你负责的产品或工序:

  • 这个产品的主要材料是什么?材料成本占比大概多少?
  • 哪些人工是直接人工,哪些是间接人工?
  • 哪些费用是固定的,哪些是变动的?

带着问题去思考,学习效果会更好!
