模块 6

成本分析方法

让数据说话——从成本数据中发现管理价值

一、为什么要做成本分析?

📖 成本分析的意义

成本核算告诉我们"花了多少钱"，成本分析则要回答：

  • 钱花在哪里了？（成本构成分析）
  • 花得对不对？（与标准、目标对比）
  • 有没有异常？（异常识别）
  • 如何改进？（发现降本机会）

成本分析是把数字变成管理决策的过程！

💼 实际场景

场景1：6206轴承成本19.30元，这个数字本身意义不大。

通过分析：

  • 材料占比65%，人工17%，费用18% → 了解成本结构
  • 与标准成本19.80元对比，低了0.50元 → 成本控制良好
  • 与去年同期19.60元对比，降低了0.30元 → 持续改善
  • 与同行业20.50元对比，有竞争优势 → 市场定位

得出结论：这个产品成本控制良好，有价格竞争力！

二、成本构成分析（料工费占比）

2.1 料工费占比分析

料工费占比计算公式
材料占比 = 直接材料成本 ÷ 产品总成本 × 100%
人工占比 = 直接人工成本 ÷ 产品总成本 × 100%
费用占比 = 制造费用 ÷ 产品总成本 × 100%

材料占比 + 人工占比 + 费用占比 = 100%

产品 材料成本 占比 人工成本 占比 制造费用 占比 总成本
6206

分析结论：

  • 6206标准品：材料占比65%，属于材料密集型，降本重点是材料采购和损耗控制
  • 6306中型：材料占比68%，规格大材料多，符合预期
  • 定制轴承：人工占22%、费用占33%，属于加工密集型，降本重点是提高效率、减少换产
改善建议：
  • 标准品：批量采购钢材争取价格优惠，降低损耗率
  • 定制品：提高批量、减少换产次数，优化工艺降低工时

2.2 行业基准对比

产品类型 材料占比 人工占比 费用占比 特征
大批量标准品 65-75% 10-15% 15-20% 自动化程度高
中批量常规品 60-70% 15-20% 15-20% 半自动化
小批量定制品 40-50% 20-30% 30-40% 手工操作多
占比异常预警：
  • 材料占比 > 80% → 可能存在材料浪费或价格过高
  • 人工占比 > 35% → 可能效率低下或工艺复杂
  • 费用占比 > 45% → 可能设备利用率低或产量不足

三、成本差异分析

3.1 标准成本与实际成本差异

📖 什么是成本差异?

成本差异 = 实际成本 - 标准成本

  • 差异为负（实际<标准）→ 成本节约，有利差异
  • 差异为正（实际>标准）→ 成本超支，不利差异

3.2 材料成本差异分析

材料成本差异分解
材料成本差异 = 价格差异 + 用量差异

价格差异 = (实际单价 - 标准单价) × 实际用量
用量差异 = (实际用量 - 标准用量) × 标准单价

📊 案例：6206轴承材料成本差异分析

GCr15钢材数据：

  • 标准单价：12.50元/kg
  • 实际单价：13.00元/kg
  • 标准用量：0.42kg
  • 实际用量：0.44kg
计算过程：

标准成本 = 0.42 × 12.50 = 5.25元
实际成本 = 0.44 × 13.00 = 5.72元
成本差异 = 5.72 - 5.25 = +0.47元（不利）

分解差异：
价格差异 = (13.00 - 12.50) × 0.44 = +0.22元（不利）
用量差异 = (0.44 - 0.42) × 12.50 = +0.25元（不利）
合计 = 0.22 + 0.25 = 0.47元

差异原因分析：

差异项 金额 可能原因 责任部门 改进措施
价格差异 +0.22元 钢材市场价上涨 采购部 锁价合同、批量采购
用量差异 +0.25元 加工损耗增加 生产部 优化工艺、提高技能

3.3 人工成本差异分析

人工成本差异分解
人工成本差异 = 工资率差异 + 效率差异

工资率差异 = (实际工资率 - 标准工资率) × 实际工时
效率差异 = (实际工时 - 标准工时) × 标准工资率

📊 案例：车削工序人工成本差异

数据：

  • 标准工资率：35元/小时
  • 实际工资率：38元/小时
  • 标准工时：0.06小时
  • 实际工时：0.07小时
标准人工成本 = 0.06 × 35 = 2.10元
实际人工成本 = 0.07 × 38 = 2.66元
人工成本差异 = 2.66 - 2.10 = +0.56元（不利）

工资率差异 = (38 - 35) × 0.07 = +0.21元
效率差异 = (0.07 - 0.06) × 35 = +0.35元
合计 = 0.21 + 0.35 = 0.56元

原因分析：

  • 工资率差异+0.21元：工资上调（人力成本上升趋势）
  • 效率差异+0.35元：实际工时超标准16.7%，可能是设备老化或工人技能不足
改进措施：
  • 工资率：属于市场因素，可接受；但要通过效率提升抵消
  • 效率：加强技能培训，设备保养，优化工艺参数

四、异常成本识别

4.1 异常成本的定义

📖 什么是异常成本?

异常成本是指明显偏离正常水平、需要特别关注和分析的成本。

识别标准：

  • 超出标准成本±10%
  • 料工费占比异常（偏离行业标准）
  • 环比、同比波动过大（>20%）
  • 工序成本明显异常

4.2 异常识别规则

异常类型 识别规则 可能原因
总成本异常 实际成本超标准成本±10% 材料涨价、效率下降、设备故障
材料占比异常 材料占比>75%或<50% 浪费、价格变动、核算错误
人工占比异常 人工占比>30%或<10% 效率低、工艺问题、工资异常
费用占比异常 费用占比>40%或<10% 产量低、分摊不合理
工序成本异常 单工序超标准>15% 该工序设备、人员、物料问题
波动异常 环比变动>20% 季节性因素、偶发事件

4.3 异常成本分析案例

🚨 案例：发现6406轴承成本异常

数据对比：

项目 本月实际 标准成本 上月实际 差异
材料成本 28.50 25.00 24.80 +14.0%
人工成本 4.50 4.20 4.10 +7.1%
制造费用 7.60 7.00 6.90 +8.6%
总成本 40.60 36.20 35.80 +12.2%

异常识别：

  • 总成本超标准12.2% → 异常！
  • 材料成本超标准14.0% → 重点关注！

深入分析：

材料 标准用量 实际用量 标准单价 实际单价 原因
GCr15钢材 0.80kg 0.90kg 12.5元/kg 13.2元/kg 用量+12.5%，价格+5.6%

追查原因：

  1. 调阅生产记录，发现磨削工序废品率高达8%（正常2%）
  2. 询问车间，磨床精度下降，需要增加磨削余量
  3. 钢材价格上涨5.6%，市场因素
改进措施：
  • 立即行动：安排磨床大修，恢复精度
  • 短期措施：加强首件检验，减少废品
  • 中期措施：优化工艺，降低磨削余量
  • 材料价格：与供应商谈判锁价

预期效果：改进后材料用量恢复至0.82kg，成本降至37.50元，接近标准。

五、成本对标分析

5.1 什么是成本对标?

📖 成本对标

成本对标是将本企业的成本与标杆（内部标杆、竞争对手、行业最佳）进行比较，找出差距和改进方向。

5.2 对标对象选择

对标类型 对标对象 数据来源 适用场景
内部对标 公司内不同车间、不同时期 内部数据 查找内部差距
竞争对标 同行业竞争对手 市场调研、产品拆解 了解竞争地位
行业对标 行业平均水平 行业报告、协会数据 行业定位
最佳实践对标 世界一流企业 公开数据、学习交流 学习先进

5.3 对标分析案例

📊 案例：6206轴承成本对标分析

项目 瓦轴 国内竞争对手A 国际标杆SKF 差距
材料成本 12.50 12.00 11.50 -8.0%
人工成本 3.20 3.50 2.00 +37.5%
制造费用 3.50 3.80 2.80 +20.0%
总成本 19.20 19.30 16.30 +15.1%

对标结论：

  • 与国内竞争对手基本持平（差距0.5%）
  • 与国际标杆有15.1%差距
  • 材料成本差距8%，主要是规模采购优势
  • 人工成本差距37.5%，主要是自动化水平差距
改进方向：
  • 材料：扩大采购批量，集中采购降低价格
  • 人工：推进自动化改造，提高劳动生产率
  • 制造费用：提高设备利用率，优化能源管理
  • 目标：3年内缩小与SKF的差距至10%以内

六、成本改善机会分析

6.1 降本增效的突破口

成本改善优先级矩阵

改善领域 成本占比 改善空间 实施难度 优先级
材料采购成本 高(65%) 中(5-8%) 低 ★★★★★
材料损耗率 高(影响材料) 中(2-5%) 中 ★★★★
人工效率 中(17%) 高(10-20%) 中 ★★★★
自动化改造 高(降人工) 高(30%+) 高 ★★★
能源管理 低(5%) 中(10-15%) 低 ★★

优先级说明：重点抓"高占比+低难度"的项目，快速见效； "高难度+高收益"的项目作为中长期规划。

6.2 具体改善措施

材料成本改善（占比65%，优先级最高）
  • 集中采购：年用量1000吨以上的钢材，集中招标采购，预计降价3-5%
  • 国产替代：部分进口辅料用国产品牌替代，降低10-15%
  • 损耗控制：优化工艺，损耗率从5%降至4%，节约材料1%
  • 预期节约：材料成本降低5%，总成本降低3.25%
人工效率提升（占比17%）
  • 技能培训：车工、磨工技能提升，工时效率提高10%
  • 工艺优化：减少不必要的工序，合并部分工序
  • 标准化作业：推行作业标准，减少变动
  • 预期节约：人工成本降低8%，总成本降低1.36%
制造费用控制（占比18%）
  • 提高产能利用率：从75%提升至85%，摊薄固定费用
  • 节能降耗：优化热处理工艺，电费降低8%
  • 预防性维护：减少设备故障，维修费降低15%
  • 预期节约：制造费用降低10%，总成本降低1.80%

七、本模块要点总结

📚 核心知识点回顾

  • 成本构成分析：料工费占比，了解成本结构，找到重点
  • 成本差异分析：实际vs标准，分解为价格差异和数量差异
  • 异常成本识别：总成本±10%、占比异常、波动过大
  • 成本对标分析：与内部、竞争对手、行业标杆对比
  • 改善机会分析：抓"高占比+低难度"的项目优先改善
  • 分析工具：对比分析、差异分析、结构分析、趋势分析
  • 最终目的：从数据中发现问题，找到改善方向，降本增效
成本分析的"四问"框架：
  1. 钱花在哪里了？→ 成本构成分析（料工费占比）
  2. 花得对不对？→ 成本差异分析（与标准对比）
  3. 有没有异常？→ 异常成本识别（超出合理范围）
  4. 如何改进？→ 改善机会分析（降本增效方向）

每月成本分析报告，都应该回答这四个问题！
