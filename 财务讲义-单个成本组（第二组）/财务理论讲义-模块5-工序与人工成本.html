模块 5

工序与人工成本

基于工序的成本核算——打通工艺与财务

一、什么是工序成本?

📖 工序成本的定义

工序成本是指产品在某道工序加工过程中发生的成本，包括该工序的人工成本和制造费用。

核心思想：产品每经过一道工序，就增加一次成本。

🏭 轴承加工的工序成本

6206轴承内圈的加工路线：

原材料 (GCr15钢棒)
   成本：1.58元
   ↓
工序1：锻造
   人工：0.25元  +  制造费用：0.30元  =  工序成本：0.55元
   累计成本：1.58 + 0.55 = 2.13元
   ↓
工序2：车削
   人工：2.04元  +  制造费用：2.80元  =  工序成本：4.84元
   累计成本：2.13 + 4.84 = 6.97元
   ↓
工序3：热处理
   人工：4.27元  +  制造费用：5.50元  =  工序成本：9.77元
   累计成本：6.97 + 9.77 = 16.74元
   ↓
工序4：磨削
   人工：6.67元  +  制造费用：8.80元  =  工序成本：15.47元
   累计成本：16.74 + 15.47 = 32.21元
   ↓
完工内圈
   总成本：32.21元

关键理解：每道工序都在"添砖加瓦"，累积形成最终成本！

二、直接人工成本核算

2.1 直接人工成本的计算公式

直接人工成本核心公式
直接人工成本 = Σ(工序标准工时 × 小时工资率)

分解为：
工序人工成本 = 工序工时 × 工序工资率

产品总人工成本 = 所有工序人工成本之和

2.2 工时的确定

2.2.1 什么是工时?

📖 工时的定义

工时是指生产一个产品或完成一道工序所需要的时间。

单位：小时(h)或分钟(min)

2.2.2 工时的三种类型

工时类型 说明 应用场景
标准工时 工艺文件规定的理论工时 标准成本核算、定额管理
实际工时 实际生产中消耗的工时 实际成本核算、绩效考核
定额工时 标准工时+宽放时间 计件工资、生产计划

2.2.3 标准工时的构成

标准工时构成
标准工时 = 作业时间 + 宽放时间

作业时间 = 主时间 + 辅助时间
- 主时间：机器实际加工时间
- 辅助时间：装卸、测量、移动时间

宽放时间包括：
- 生理需要时间（上厕所、喝水）
- 准备结束时间（领料、交接班）
- 休息时间（疲劳恢复）

一般：宽放率 = 10-15%

⏱️ 案例：计算车削工序的标准工时

测定数据：

  • 机器实际加工时间（主时间）：2.8分钟
  • 装卸工件时间：0.4分钟
  • 测量尺寸时间：0.2分钟
  • 宽放率：10%
计算过程：

辅助时间 = 0.4 + 0.2 = 0.6分钟
作业时间 = 2.8 + 0.6 = 3.4分钟
宽放时间 = 3.4 × 10% = 0.34分钟
标准工时 = 3.4 + 0.34 = 3.74分钟 ≈ 3.5分钟

2.2.4 工时的确定方法

确定方法 说明 优缺点
技术测定法 用秒表实测，进行动作分析 ✓准确 ✗耗时
统计分析法 统计历史实际工时的平均值 ✓简单 ✗受异常影响
经验估算法 参考类似产品的工时 ✓快速 ✗不够精确
定额手册法 查阅行业或企业标准定额手册 ✓标准化 ✗通用性强但针对性弱
课题建议：
  • 优先使用工艺文件中的标准工时
  • 如果没有标准工时，可以从MES系统提取历史实际工时，取平均值
  • 新产品可以参考类似产品的工时，试生产后再调整
  • 工时定额应定期回顾修订（建议每年一次）

2.3 工资率的计算

2.3.1 小时工资率公式

小时工资率计算
小时工资率 = (工资总额 + 社保公积金) ÷ 有效工作小时数

月有效工作小时 = 月工作天数 × 每天工作小时
一般：22天 × 8小时 = 176小时/月

💰 案例：计算某车间的小时工资率

数据：

  • 车间直接生产工人：50人
  • 月工资总额：25万元
  • 社保公积金：6.25万元（工资的25%）
  • 月工作天数：22天
  • 每天工作时间：8小时
第一步：计算总人工成本
总人工成本 = 25万 + 6.25万 = 31.25万元

第二步：计算总工时
月有效工时 = 22天 × 8小时 = 176小时/人
总工时 = 50人 × 176小时 = 8,800小时

第三步：计算小时工资率
小时工资率 = 312,500元 ÷ 8,800小时
= 35.5元/小时

应用：某产品在该车间加工0.5小时，人工成本 = 0.5 × 35.5 = 17.75元

2.3.2 按工序确定工资率

不同工序的技术难度、工资水平不同，可以分工序计算工资率：

工序 技能要求 平均工资(元/月) 月工时(h) 工资率(元/h)
锻造 中等 6,000 176 34
车削 较高 7,000 176 40
热处理 中等 6,500 176 37
磨削 高 8,000 176 45
装配 中等 6,500 176 37

注：工资率已包含社保公积金(按工资的25%计算)

三、工序制造费用核算

3.1 工序制造费用的分配

📖 工序制造费用

工序制造费用是指该工序发生的间接费用，如设备折旧、能源、维修等。

分配原理：按照产品在该工序消耗的作业动因分配费用。

3.2 工序费用分配率的计算

工序费用分配率
工序费用分配率 = 工序制造费用总额 ÷ 工序作业动因总量

产品分配的工序费用 = 产品消耗的动因量 × 分配率

💡 案例：计算磨削工序的费用分配率

磨削工序月度数据：

  • 设备折旧：8万元
  • 电费：3万元
  • 维修费：2万元
  • 间接人工：1.5万元
  • 其他费用：0.5万元
  • 制造费用合计：15万元

作业动因：机器工时，月度总工时1,500小时

计算费用分配率：

磨削工序费用分配率 = 150,000元 ÷ 1,500小时
= 100元/小时

应用：
6206内圈磨削工时 = 0.0833小时
分配的制造费用 = 0.0833 × 100 = 8.33元

五、本模块要点总结

📚 核心知识点回顾

  • 工序成本：产品每经过一道工序，就增加人工成本和制造费用
  • 人工成本公式：工序工时 × 工序工资率
  • 工时类型：标准工时、实际工时、定额工时（课题建议用标准工时）
  • 工资率计算：(工资+社保) ÷ 有效工时，可按车间或工序分别计算
  • 工序费用分配：按作业动因（机器工时、人工工时等）分配
  • 成本累积：材料成本→工序1→工序2→...→最终产品成本
  • 数据来源：工时来自工艺文件或MES，工资来自人力和财务
课题实施要点：
  1. 从工艺部获取完整的工艺路线和工时定额
  2. 从人力资源部和财务部获取工资数据，计算工资率
  3. 将制造费用按工序归集，计算各工序的费用分配率
  4. 按工序逐步计算，建立工序成本累积表
  5. 定期对比标准工时与实际工时，发现效率改进机会
